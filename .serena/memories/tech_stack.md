# Technology Stack

## Core Technologies
- **Markdown**: Primary document format (.md files)
- **PDF Export**: Final deliverable format
- **Bash**: Shell scripting for automation
- **Git**: Version control

## AI & MCP Integration
- **Gemini CLI**: AI-powered document generation and editing
- **MCP Servers**:
  - `sequential-thinking-mcp`: Complex reasoning and analysis
  - `serena`: Project memory and session management
- **Python Virtual Environment** (.venv): For MCP server dependencies

## Claude Code Integration
- **Custom Commands**: 
  - `/doc-design`: Professional document design analysis with Korean typography expertise
- **Claudedocs**: Documentation and usage guides stored in `claudedocs/`

## Document Production
- **Fonts**: KoPub World fonts (바탕체, 돋움체) for professional Korean typography
- **Styles**: Custom styling in `styles/` directory
- **Media**: Supporting images and graphics in `media/` directory

## Platform
- **macOS** (Darwin system) - affects command availability and paths
