# Document Style & Conventions

## File Naming Conventions
- **Descriptive names**: Use clear, descriptive names (e.g., `Hakone_Retreat_Fore_Comparison.md`)
- **Underscores**: Use underscores `_` for spaces in filenames
- **Capitalization**: Title Case for document names
- **Extensions**: `.md` for source, `.pdf` for final deliverables

## Korean Typography Standards

### Font Recommendations
- **Body Text (본문)**: KoPub World 바탕체 (Batang) - serif, digital-optimized
- **Headings (제목)**: KoPub World 돋움체 (Dotum) - sans-serif
- **Font Sources**: https://www.kopus.org/biz-electronic-font2/ (free, commercial use allowed)

### Typography Rules
- **Font Size**: Korean 13pt = English 12pt (Korean needs +1-2pt)
- **Line Height**: Minimum 1.6, recommended 1.7-1.8 for Korean text
- **Letter Spacing (자간)**: 0 for body, +5 for headlines
- **Character Width (장평)**: 95-105% range
- **Never use italic** for Korean text emphasis (use bold or color instead)

### Mixed-Language Documents
- **Font Pairing**: KoPub World + Inter or Noto fonts
- **Line Height**: 1.7 minimum to accommodate Korean height
- **Word Spacing**: 0.05em slight spacing between scripts
- **Language Tags**: Add for screen reader accessibility

## Document Structure
- **Formal Tone**: Professional, respectful language for academic audience
- **Clear Hierarchy**: H1 > H2 > H3 structure
- **Page Numbers**: Required for professional documents
- **Table of Contents**: For documents >3 pages

## Content Conventions
- **Dates**: Format as "January 15-17, 2026" or "2026년 1월 15-17일"
- **Currency**: Korean Won (₩) for budgets
- **Comparisons**: Use tables for side-by-side analysis
- **Evidence**: Include specific numbers, percentages, concrete data
