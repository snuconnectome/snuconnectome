# MacBook Pro에서 PDF 다운로드

## 가장 간단한 방법

MacBook Pro의 터미널에서 다음 명령 실행:

```bash
cd ~/Desktop
curl -L -o "워크샵_제출_자료.pdf" \
  "https://github.com/snuconnectome/snuconnectome/raw/main/워크샵_제출_자료.pdf"
```

## 또는 브라우저에서

1. 다음 URL 접속:
   https://github.com/snuconnectome/snuconnectome/raw/main/워크샵_제출_자료.pdf

2. 브라우저가 자동으로 다운로드하거나, 우클릭 → "다른 이름으로 저장" → Desktop 선택

## GitHub 웹사이트에서

1. https://github.com/snuconnectome/snuconnectome 접속
2. `워크샵_제출_자료.pdf` 파일 클릭
3. "Download" 버튼 클릭
4. Desktop으로 이동
