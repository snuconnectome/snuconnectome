# Hands-on Agentic AI 워크샵 일정 구성 프롬프트

## 워크샵 개요

### 기본 정보
- **워크샵 명칭**: Hands-on Agentic AI 워크샵
- **기간**: 1일 (하루 전체)
- **장소**: 일본 하코네
- **주최**: 차지욱 교수 (서울대학교 심리학과) + Isomura 박사 (RIKEN)
- **대상**: 심리학과 교수진 및 연구자

### 워크샵 목적
- Agentic AI의 이론적 기초와 실무 적용 방법 학습
- Hands-on 실습을 통한 실제 구현 경험
- 심리학 연구에 Agentic AI를 적용하는 방법 탐구
- 차지욱 교수와 Isomura 박사의 전문 지식 공유

## 일정 구성 원칙

### 1. 시간 배분
- **총 시간**: 하루 (약 8-9시간)
- **이론 강의**: 30-40% (약 3시간)
- **실습 (Hands-on)**: 50-60% (약 4-5시간)
- **토론 및 질의응답**: 10-20% (약 1시간)
- **휴식 시간**: 점심, 커피 브레이크 포함

### 2. 세션 구성 요소

#### 오전 세션 (이론 및 기초)
- **Agentic AI 개론** (60-90분)
  - Agentic AI의 정의와 개념
  - 기존 AI와의 차이점
  - Agentic AI의 핵심 구성 요소 (Planning, Tool Use, Memory, Reflection)
  - 심리학 연구에의 적용 가능성

- **기술적 기초** (60-90분)
  - LLM 기반 Agent 아키텍처
  - ReAct (Reasoning + Acting) 패러다임
  - Tool Calling 및 Function Calling
  - Memory 및 Context Management

#### 오후 세션 (실습 중심)
- **Hands-on 실습 1: 기본 Agent 구축** (90-120분)
  - 간단한 Agent 구현 실습
  - Tool 사용 예제
  - 기본적인 Planning 구현

- **Hands-on 실습 2: 고급 Agent 구현** (90-120분)
  - Multi-step Planning
  - Memory 활용
  - Reflection 및 Self-correction
  - 심리학 연구 시나리오 적용

- **프로젝트 워크샵** (60-90분)
  - 참석자별 프로젝트 아이디어 발굴
  - 실제 연구에 적용 가능한 Agent 설계
  - 그룹별 토론 및 피드백

#### 마무리 세션
- **종합 토론 및 Q&A** (30-60분)
  - 실습 결과 공유
  - 질의응답
  - 향후 연구 방향 논의

### 3. 실습 환경
- **필요 도구**:
  - Python 환경 (Jupyter Notebook 또는 Google Colab)
  - LLM API 접근 (OpenAI, Anthropic 등)
  - 기본 라이브러리: langchain, langgraph, 또는 직접 구현
- **난이도**: 초급-중급 (프로그래밍 기본 지식 필요)
- **준비물**: 노트북 (참석자 개인 준비)

### 4. 강사 역할 분담

#### 차지욱 교수
- Agentic AI의 심리학적 관점 및 응용
- 심리학 연구 시나리오 제시
- 실습 프로젝트 멘토링

#### Isomura 박사
- Agentic AI의 기술적 기초 및 최신 동향
- RIKEN에서의 연구 경험 공유
- 고급 구현 기법 및 Best Practices

### 6. 세부 내용 작성 가이드

각 세션의 "내용" 컬럼에는 다음 정보를 포함:
- 주요 주제 및 학습 목표
- 다룰 내용의 핵심 키워드
- 실습이 포함된 경우 실습 내용 요약
- 예상 결과물 (있는 경우)

예시:
- "Agentic AI 개론 및 심리학적 관점<br>- Agentic AI의 정의와 핵심 개념<br>- Planning, Tool Use, Memory, Reflection<br>- 심리학 연구에의 적용 가능성 탐구<br>- 실습 환경 설정"

### 7. 표 형식 요구사항

- 마크다운 표 형식 사용
- 각 셀은 `<br>` 태그로 줄바꿈 가능
- **볼드**는 `**텍스트**` 형식 사용
- 표는 HTML로 변환 시 제대로 렌더링되도록 작성

### 8. 전체 일정 통합

- **1일차**: 학사 협의회 (기존 일정)
- **2일차**: Hands-on Agentic AI 워크샵 (새로 추가)
- **3일차**: 선택 사항 (자유 시간 또는 추가 토론)
